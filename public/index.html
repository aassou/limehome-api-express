Welcome on Limehome challenge task Express API!

Here is a list of some routes:

  • HTTP Method: GET -> Get all reservations: /reservation
  • HTTP Method: GET -> Get a reservation with id: /reservation/:id
  • HTTP Method: POST -> Add a reservation: /reservation
  • HTTP Method: PUT -> Update a reservation: /reservation/:id
  • HTTP Method: DELETE -> Delete a reservation: /reservation/:id
